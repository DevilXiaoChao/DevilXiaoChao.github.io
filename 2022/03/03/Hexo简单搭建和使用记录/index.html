1.安装Git

2.安装NodeJs

  • 下载：NodeJs

  • 修改 npm 的安装源为淘宝NPM镜像

3.安装Hexo

4.GitHub创建个人仓库

  • 新建一个repository
  • 命名：<你的 GitHub 用户名>.github.io

5.连接GitHub

  • 打开Git Bash
  • 创建SSH，一路回车
  • 查看SSH Key

  • 打开GitHub，点击自己的头像，再点击Settings

  • 点击SSH and GPG keys

  • 点击New SSH key

输入SSH key的名字，随便取；

再输入刚才复制的SSH key；

  • 点击Add SSH key完成添加
